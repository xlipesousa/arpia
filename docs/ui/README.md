1. Substituir `hunt-blue-red-wireframes.png` pelo arquivo exportado do Figma/Sketch mantendo o mesmo nome.
2. Adicionar a fonte editável (`.sketch`, `.fig` ou `.xd`) com versionamento adequado; preservar o nome atual.
3. Registrar no `docs/hunt-phase3-modelagem.md` a data de recebimento e qualquer guideline relevante (cores, spacing, ícones).
